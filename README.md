# MassbitRoute Stat Component

## Installation
```
export GIT_PRIVATE_READ_URL="http://massbit:xxx@git.massbitroute.dev"
mkdir -p /massbit/massbitroute/app/src/sites/services/stat
git clone https://github.com/massbitprotocol/massbitroute_stat.git /massbit/massbitroute/app/src/sites/services/stat -b dev
/massbit/massbitroute/app/src/sites/services/stat/scripts/run _install
```
